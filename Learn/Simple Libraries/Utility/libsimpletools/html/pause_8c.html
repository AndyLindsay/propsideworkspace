void pause ( int  dt)

Delay cog from moving on to the next statement for a certain amount of time.

The defualt time increemnt is 1 ms, so pause(100) would delay for 100 ms = 1/10th of a second. This time increment can be changed with a call to the setPauseDt function.

Parameters
dtThe number of time increments to delay.
