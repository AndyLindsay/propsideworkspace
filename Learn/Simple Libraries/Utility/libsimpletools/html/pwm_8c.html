Use an I/O pin to transmit PWM signal.

Typically used for signals that control DC motors. Default time increments are specified in 1 microsecond units. Unit size can be changed with a call to setIoDt funciton.

Parameters
tHightime the output signal stays high
tCycletime of the cycle (high time + low time)
