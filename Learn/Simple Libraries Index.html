Simple Libraries Folder Index

Here are links of Simple Libraries that come with this Learn folder:

Utility

___Simple Tools - Tools that simplify I/O, timing, memory, and communication

Audio

___libwavplayer - Plays 16-bit, 32 ksps, mono .wav files (Propeller Activity Board/Propeller BOE).

Convert

___libadcDCpropAB - Propeller Activity Board A/D conversion for voltage measurements

___libadcACpropAB - Propeller Activity Board A/D conversion for signal measurements

___libdacctr - Propeller Activity Board D/A for setting voltages

Motor

___libservo - Servo control library

___libservodiffdrive - Convenience functions for robot control with differential drive servos

___For DC motor control, see ...Documents/SimpleIDE/Learn/Examples/Devices/Motor/DC.

Sensor

___libcompass3d - For measurements with the 3-axis HMC5883L compass.

___libmx2125 - For measurements with the 2-axis mx2125 accelerometer.

___libping - For measurements with the Ping))) Ultrasonic Distance Sensor.
